¿Quieres contribuir al desarrollo de RoboTito?

Bueno, ¡nos alegra!. Actualmente RoboTito es solo un proyecto personal, pero no vamos a despreciar ningún tipo de apoyo.

Vamos a ver que puedes hacer.

GitHub

Desarrollador

Casi todo el código de RoboTito (menos excepciones específicas) se encuentra en GitHub, una de las plataformas más populares sino la más popular para asentar proyectos.

Esta es la manera más directa de contribuir al proyecto, puesto que vas a trabajar sobre el código que le da vida a RoboTito, proponiendo funciones, características y solucionando problemas.

Las instrucciones para comenzar a desarrollar están en el inicio delrepositorioen GitHub.

Reportes

Quizá no sepas programar o no te interese, ¡está perfecto!, puedes hacer más cosas por RoboTito que solo escribir código y plantear novedades.

GitHub ofrece una sección donde puedes incluir reportes de errores, sugerencias de cambios o nuevas funcionalidades. Si quieres hacerlo, puedes encontrar la sección de issues en el repositorio enGitHubo también la sección de pulls en estapágina.

Discord

Como método de comunicación (como parecía claro) utilizamos Discord,ya que nos permite ser rápidos para realizar pruebas sobre el bot y para comunicarnos fácilmente a través de canales de texto o de voz.

Ahí vas a poder hablar directamente con nosotros y probar el bot tanto como quieras, a la vez que pasar un buen rato, como en cualquier servidor de Discord.

RoboTito

Un bot multipropósito para Discord.

RoboTito, de la unión de Robo (Robot) y Tito (el apodo del creador), es un bot para Discord que se enfoca en abarcar múltiples funcionalidades, desde ayudar a la moderación de un servidor hasta el entretenimiento y la interacción de sus integrantes.
